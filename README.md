# ENGCC304

### คำอธิบายรายวิชา
ศึกษาและฝึกปฏิบัติเกี่ยวกับแนวคิดและองค์ประกอบของระบบคอมพิวเตอร์ ปฏิสัมพันธ์ (Interaction) ระหว่างฮาร์ดแวร์และซอฟต์แวร์ ภาษาที่ใช้เขียนโปรแกรมในปัจจุบัน

### เกณฑ์การให้คะแนน
* เข้าห้องเรียน 10%
* การบ้าน 10%
* ปฏิบัติ (Lab) 20%
* สอบปฏิบัติ 20%
* สอบกลางภาค (Midterm) 20%
* สอบปลายภาค (Final) 20%
